Thank you for downloading **Kaiserreich** for **Hearts of Iron IV**.
This release is built for **Hearts of Iron IV version 1.17.4**.
Compatibility with subsequent hotfixes is expected but not guaranteed.

## Manual Installation Instructions
1. Download the mod archive.
2. Extract the .zip file to `Documents\Paradox Interactive\Hearts of Iron IV\mod`.
3. In the extracted folder:
   - Make a copy of `descriptor.mod`.
   - Edit the file if necessary to ensure the `name` and `path` fields match the folder name.
   - Rename and move the `.mod` file to `Hearts of Iron IV\mod`.
4. Enable the mod in the Launcher.

Detailed step-by-step instructions are available on the Kaiserreich Wiki:
https://kaiserreich.wiki/Manual_installation
